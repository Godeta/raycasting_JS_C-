Raycasting Project

This project (not finished just an idea that I started on a whim) is inspired by various works, including:

  • The game Wolfenstein 3D (Wikipedia)
  • A video on implementing raycasting in C (YouTube)
  • A tutorial on raycasting in C++ (YouTube)
  • Daniel Shiffman's work (YouTube)

These resources have contributed to the development of this raycasting project, which demonstrates 3D-like rendering using 2D techniques.

To see the raycasting demo in action, please visit the Raycast JS Demo.
